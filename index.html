California State Parks with surrounding Campgrounds, Toilets, and Recreational Routes

Authored by: Ashley Ly

California's 88 State Parks

California State Parks preserves the state's biological diversity and creates opportunities for outdoor recreational activities for anybody! California State Park can be visited year-round with diverse environments with any activity whether its hiking, biking, or just relaxing. This map displays all 88 state parks along with surrounding campgrounds, toilets, and recreational routes- the main necessities when visiting a state park. In the map, the state park entry points are labeled by the state park name and the state park recreational route is labeled by trail name.

Data used in this project

Park Entry Points, Campgrounds, Buildings, and Park Routes data was collected from California State Parks GIS Datasets published by California Department of Parks and Recreation. Data accessed December, 2020. The data collected was then parsed and filtered through for only state parks, public use campground, all types of toilets and state park routes.

Data is shown on Mapbox web map.
